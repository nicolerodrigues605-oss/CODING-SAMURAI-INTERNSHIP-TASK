# Titanic EDA Project

## Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python.

## Objectives
- Analyze passenger survival trends
- Handle missing values
- Create visualizations
- Identify important patterns in the dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Titanic dataset from Seaborn library.

## Key Analysis
- Missing Value Analysis
- Survival Analysis
- Gender-based Analysis
- Passenger Class Analysis
- Correlation Heatmap
- Histograms and Countplots

## Key Findings
- Female passengers had higher survival rates
- First-class passengers survived more often
- Younger passengers had slightly better survival chances
- Passenger class influenced survival significantly
